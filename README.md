# php-coursework2-normalization-sql

# Coursework 2: Normalization and SQL Implementation

This repository contains the solution for Coursework 2, focusing on database normalization (3NF) and SQL implementation in MySQL

## Files Description

* **`normalization_process.pdf`**: step-by-step process for converting the schema to 1NF, 2NF, and 3NF
* **`schema.sql`**: Contains all `CREATE TABLE` statements with appropriate PRIMARY KEY, FOREIGN KEY, and other constraints
* **`data.sql`**: Contains all `INSERT INTO` statements used to populate the tables with data (30+ records total), ensuring referential integrity. It also includes 2 example SELECT queries per table
* **`queries.sql`**: Contains all the required advanced `SELECT` statements

## How to import the dabatase
1. Import `schema.sql`;
2. Import `data.sql`
